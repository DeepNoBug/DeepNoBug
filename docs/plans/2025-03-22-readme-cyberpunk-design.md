## 区域设计

### 1. Header 区域
- **组件:** capsule-render (rounded + 扫描线动画)
- **打字效果:** typing-svg (青色发光)
- **头像:** 全息光环效果

### 2. 技术栈区域
- **技能条:** 霓虹进度条 + 动态填充
- **工具网格:** 全息图标，悬停发光
- **安全工具:** 武器库卡片布局

### 3. 项目展示区域
- **卡片样式:** 半透明玻璃 + 青色发光边框
- **标签:** 霓虹胶囊样式
- **按钮:** 赛博朋克风格 [▶ ACCESS]
- **悬停效果:** 上浮 + 光晕增强

### 4. 统计数据区域
- **Stats 图表:** 自定义 cyber-blue 主题
- **Streak:** 蓝青配色
- **热力图:** 青色到紫色渐变
- **活动图:** 发光曲线

### 5. 页脚区域
- **联系方式:** 霓虹按钮
- **引用:** 滚动打字效果
- **标语:** "The code is law. The terminal is truth."
